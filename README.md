## How to subscribe

### Apple Calendar
1. Open Calendar.
2. Choose **File** or **Calendar** → **New Calendar Subscription**.
3. Paste the feed URL.
4. Save.

### Google Calendar
1. Open Google Calendar.
2. Next to **Other calendars**, choose **Add by URL**.
3. Paste the feed URL.
4. Add calendar.

## Automatic updates

These feeds are refreshed from Dribl on:
- **Monday 9:00am** Australia/Sydney
- **Wednesday 9:00am** Australia/Sydney

If fixtures change in Dribl, the feed is republished to the same URL.

## Notes

- Keep using the same URL, subscriber calendars refresh automatically.
- Refresh timing is controlled by Apple/Google Calendar, not by GitHub.
- The feeds are intentionally kept separate, not combined.
